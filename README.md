## Overview
- poseidon独立的日志记录中心
- 需要配合poseidon版本2.0.0及以上
- poseidon(https://github.com/cwdtom/poseidon)
    
## Usage
- 启动jar包
    ```shell
      java -jar poseidon-center.jar
    ```
    1. -p 监听端口，默认为10000
    1. -v 日志中心版本号
    1. -c 指定自定义的logback.xml文件
    1. -h 显示帮助中心
